# TDGP
## Three Dimensional Genome Pipeline
#### my 3D genome pipeline and scripts

## Install
```bash
export PATH=/path/to/TDGP/bin:/path/to/TDGP/utils:$PATH
export PYTHONPATH=/path/to/TDGP:$PYTHONPATH
```

## Dependencies
### requirement
- pysam
- cooler
- joblib
- numpy = 1.16.1
- matplotlib
- pandas
- pyranges
- seaborn >= 0.9.0
- snakemake
- xlwt
### optional
- pyGenomeTracks
- deeptools
- tadlib
